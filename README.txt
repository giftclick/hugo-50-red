SITIO ESTÁTICO DE GIFTCLICK

1. Descomprime todo el contenido sin cambiar las carpetas.
2. Sube index.html junto con las carpetas css, js y assets a tu hosting.
3. Para generar enlaces con nombre y pases, abre pases.html.
4. Para probar localmente, usa un servidor estático (por ejemplo: npx serve .).

Archivos HTML incluidos:
- index.html (index.html)
- home.html (home.html)
- pases.html (generador de enlaces personalizados)

La apariencia, imágenes, música y navegación entre partes fueron generadas desde la vista actual del editor.
Los enlaces WhatsApp de confirmación se conservan para uso en hosting estático.
Si la plantilla incluye un formulario RSVP propio de GiftClick, esa copia estática es visual y no se conecta automáticamente al backend.
